##  What This System Does

| Step | Technology | Description |
|------|-----------|-------------|
| 1 Upload | React + S3 | Drag-and-drop PDF/image upload via presigned URLs |
| 2 Store | AWS S3 + DynamoDB | File stored securely; metadata persisted |
| 3 Stream | Apache Kafka | `document.uploaded` event triggers async pipeline |
| 4 OCR | AWS Textract / Tesseract | Raw text extracted from documents |
| 5 Extract | GPT-4o / Claude 3 / Gemini | Structured JSON data extracted by LLM |
| 6 Store Results | PostgreSQL + DynamoDB | Structured fields stored for querying |
| 7 Analyze | Spring Boot Analytics Service | Metrics, trends, anomaly detection |
| 8 Visualize | React Dashboard | Charts, document viewer, confidence scores |
